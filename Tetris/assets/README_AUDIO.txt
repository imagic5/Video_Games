Audio Files Required:

Please place the following audio files in this directory (assets/):

Music:
- music.ogg (or .mp3) - Background music loop

Sound Effects:
- move.wav      - Piece movement
- rotate.wav    - Piece rotation
- drop.wav      - Hard drop / Lock
- clear.wav     - Line clear
- gameover.wav  - Game Over sound

Note: The game will run without these files, but will be silent.
